## for

- `for(i := 0; i < 10; i++)`,`for(;i<10;)`,`for(a, b := 0, 0; a < 10 && b < 6;a++,b++)`,`for {}`
- `for item in array`，`for index, item in array``
- `break`终止循环
- `continue`跳过本次循环
- `Label`标签，`break`、`continux`、`goto`使用该标签

### 乘法表

```go
for i := 1; i < 10; i++ {
  for j := 1; j <= i; j++ {
    fmt.Printf("%d * %d = %d\t", j , i, j * i)
  }
  fmt.Println()
}
```
